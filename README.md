# Book Club
Telegram bot for book clubs.


# Usage
```
export BOOKCLUB_BOTTOKEN=[your-token]
go get
go run main.go
```

### With docker
```
export BOOKCLUB_BOTTOKEN=[your-token]
docker-compose up
```

### Local development
For convenience you can copy `.env-template` to `.env` and populate your token there instead of exporting the variable.
